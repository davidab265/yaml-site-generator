## [1.0.0] - 2025-11-18

### Initial Release

#### Added
- Modern, responsive landing page with dark theme support
- YAML-based configuration system for easy customization
- Environment variable support as alternative configuration method
- Multiple logo options:
  - Emoji icons
  - Font Awesome icons (solid, brands, regular)
  - SVG icons
  - Image URLs
- Configurable grid layout (1-4 columns)
- Custom CSS classes for individual boxes
- Server-side rendering with Next.js 14
- Docker support with multi-stage build
- Docker Compose configuration for easy deployment
- Kubernetes deployment manifests:
  - ConfigMap for configuration
  - Deployment with 2 replicas
  - Service for internal access
  - Ingress for external access
- Tailwind CSS for styling
- Dark theme with custom color palette
- Responsive design for all screen sizes
- Example configurations and documentation
